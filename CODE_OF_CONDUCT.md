# Code of Conduct

We strive for a respectful, inclusive, and collaborative community. By participating, you agree to:

- Treat everyone with courtesy, empathy, and professionalism.
- Welcome diverse viewpoints and resolve disagreements constructively.
- Avoid harassment, discrimination, or exclusionary behavior.
- Protect privacy and refrain from sharing sensitive information.
- Use inclusive language and provide actionable, kind feedback.

## Unacceptable Behavior
Harassment, personal attacks, derogatory comments, intimidation, or sustained disruption of discussions are not tolerated.

## Reporting
If you experience or witness unacceptable conduct, please open a confidential issue or contact the maintainers directly. Include as much context as possible so we can investigate promptly and fairly. We will respect confidentiality requests whenever possible.

## Enforcement
Maintainers may take appropriate action (warnings, issue/PR locking, or removal from the community) to uphold this code of conduct.
